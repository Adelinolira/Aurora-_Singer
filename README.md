# Aurora-_Singer

# Sistema de verificação de Decolagem
  Este projeto simula um sistema de análise de telemetria utilizado antes da decolagem de uma nave espacial.
  Com o objetivo de verificar se os parâmetros da nave estão dentro da faixa de segurança antes de autirizar a decolagem da missão.

## Parâmetros analisados
- Temperatura interna
- Temperatura externa
- Integridade estrutural
- Níveis de energia
- Pressão dos tanques
- Status dos módulos críticos

## Funcionalidades do sistema
- Simulação de sensores com valores aleatórios
- Verificação automática das condições de segurança
- Cálculo de autonomia energética
- Classificação de risco da missão
- Decisão final: PRONTO PARA DECOLAR ou DECOLAGEM ABORTADA

## Tecnologias utilizadas
- Python
- Google Colab
- Github

## Como executar
1. Abra o arquivo `.ipynb` no Google Colab
2. Execute as células em ordem
3. O sistema irá simular a análise de uma missão espacial.
